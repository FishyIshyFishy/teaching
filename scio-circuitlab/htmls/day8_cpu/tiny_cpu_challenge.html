Instruction set

A cell's number decodes as op × 16 + address. A blank (0) cell means HALT.

0HALTstop the program
1LOAD aacc = mem[a]
2STORE amem[a] = acc
3ADD aacc = acc + mem[a]
4SUB aacc = acc − mem[a]
5JUMP ago to cell a
6JMPZ aif acc = 0, go to cell a
