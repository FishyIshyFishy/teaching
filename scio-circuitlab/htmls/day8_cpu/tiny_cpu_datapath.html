A cell is 8 bits: top 4 = opcode, bottom 4 = address.

0000HALTstop the program
0001LOAD aacc = mem[a]
0010STORE amem[a] = acc
0011ADD aacc = acc + mem[a]
0100SUB aacc = acc − mem[a]
0101JUMP ago to cell a
0110JMPZ aif acc = 0, go to cell a
