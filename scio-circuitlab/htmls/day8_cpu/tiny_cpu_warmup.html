Tiny CPU Warm-up

Memory — 16 cells (set the bits in the free cells)

Instruction set

Each cell is 8 bits: the top 4 are the opcode, the bottom 4 are the address. An all-zero cell means HALT.

0000HALTstop the program
0001LOAD aacc = mem[a]
0010STORE amem[a] = acc
0011ADD aacc = acc + mem[a]
0100SUB aacc = acc − mem[a]
0101JUMP ago to cell a
0110JMPZ aif acc = 0, go to cell a
Not checked
